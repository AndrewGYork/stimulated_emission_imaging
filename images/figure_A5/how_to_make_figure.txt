1. Run figure_A5.py in figure generation folder.

3. Export to pdf (figure_A5.pdf).
4. Copy figure_A5.pdf into the pdf2svg folder
5. In command prompt, type: pdf2svg figure_A5.pdf darkfield_nd_delay_scan.svg
6. Place darkfield_nd_delay_scan.svg back into images/figure_A5.
